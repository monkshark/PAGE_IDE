# PAGE IDE

> 다언어 데스크톱 IDE — **P**air · **A**tlas · **G**lass · **E**cho

PAGE는 한 페이지 안에서 네 가지 차원을 본다. 코드(텍스트), 코드 그래프(공간), 작업 시간축(시간), AI 동반자(대화).

- **Pair** — 곁에서 코드를 함께 보는 AI 동반자 (관찰자 / 대화 / 에이전트 / 튜터)
- **Atlas** — 모듈·함수·의존성을 노드와 엣지로 그리는 코드 그래프
- **Glass** — 글래스모피즘 기반 디자인 시스템
- **Echo** — 키스트로크를 로컬 SQLite에 저장하는 작업 타임라인
